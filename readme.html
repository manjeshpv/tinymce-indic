About PramukhIME TinyMCE Plugin

What is PramukhIME TinyMCE Plugin?

PramukhIME TinyMCE Plugin is a plugin for TinyMCE HTML editor using which any Web developer can enable the users to type in 23 Indian languages without downloading/installing any special software. It supports Assamese, Bengali, Bodo, Dogri, Gujarati, Hindi, Kannada, Kashmiri, Konkani, Maithili, Malayalam, Meitei, Manipuri, Marathi (Devanagari and Modi), Nepali, Oriya, Punjabi, Sanskrit, Santali (Ol Chiki), Sindhi, Sora, Tamil and Telugu.

Plugin Installation

  • Unzip the file on your local hard disk.
  • Upload/copy "pramukhime" folder to TinyMCE_ROOT/plugins/ so the folder becomes TinyMCE_ROOT/plugins/pramukhime/ (PRAMUKHIME_ROOT)
  • In the init function for TinyMCE in the Webpage, add "pramukhime" in a plugin list and add "pramukhime", "pramukhimesettings", "pramukhimeresetsettings", "pramukhimetogglelanguage" and "pramukhimehelp" buttons for the language dropdown list, keyboard settings, reset keyboard settings, toggle between the current and last selected language and help buttons respectively

Plugin Customization

If you want to customize a list of languages to be displayed in dropdown list, change toggle key and for advanced options, add "pramukhime_options" parameter in tinymce.init() function. Follow the steps given in example.html file.

PramukhIME TinyMCE Plugin Example

     
Note: This example uses TinyMCE CDN. As soon as you access this file from real webserver on localhost, it will work.

How to use?

  1. Open this file using real webserver like IIS/Apache/Nginx so that this webpage opens up on http://localhost domain. This example should be used only on localhost with port 80.
  2. Open this file in text editor and change the options given as per comments within html code to customize PramukhIME plugin.
This is a developer version plugin and should not be used in production environment. Please buy the license to use it in production environment.
